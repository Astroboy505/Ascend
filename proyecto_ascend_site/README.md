# Proyecto Ascend - Página promocional falsa estilo tienda

Página HTML estática preparada para GitHub Pages.

## Cómo usar
1. Sube todo el contenido de esta carpeta a un repositorio de GitHub.
2. Ve a Settings > Pages.
3. Activa GitHub Pages desde la rama `main` y carpeta `/root`.
4. Para que el botón de descarga funcione, mete tu ejecutable comprimido en `downloads/ProyectoAscend.zip`.

Esta página no es una página oficial de Steam. Incluye avisos visuales y texto indicándolo.
